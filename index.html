Open Audit

A simple tool to carry out opinion poll

Go To Open Audit

Our web application is based on simple interactive polls that can be found all over the internet.

Try out our state of the art functionalities!

Create your own rooms!

You will be able to create a room, design your own questions along with the desired number of possible answers.
After that, you'll be able to manage your room through multiple options.

Create Design Manage

Join a room and answer questions!

Select the room you want to join within the rooms list. Multiple questions will be displayed, all you need to do is answer them.

Select Answer

Awsome live pie charts!

Statistics will be performed live on all answers. Pie charts will be displayed next to each coresponding question
